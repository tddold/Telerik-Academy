    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-homework-defining-classes---part-2" class="anchor" href="#homework-defining-classes---part-2" aria-hidden="true"><span class="octicon octicon-link"></span></a>Homework: Defining Classes - Part 2</h1>

<h3>
<a id="user-content-problem-1-structure" class="anchor" href="#problem-1-structure" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 1. Structure</h3>

<ul class="task-list">
<li>  Create a structure <code>Point3D</code> to hold a 3D-coordinate {X, Y, Z} in the Euclidian 3D space.</li>
<li>  Implement the <code>ToString()</code> to enable printing a 3D point.</li>
</ul>

<h3>
<a id="user-content-problem-2-static-read-only-field" class="anchor" href="#problem-2-static-read-only-field" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 2. Static read-only field</h3>

<ul class="task-list">
<li>  Add a <code>private static read-only</code> field to hold the start of the coordinate system – the point O{0, 0, 0}.</li>
<li>  Add a static property to return the point O.</li>
</ul>

<h3>
<a id="user-content-problem-3-static-class" class="anchor" href="#problem-3-static-class" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 3. Static class</h3>

<ul class="task-list">
<li>  Write a <code>static class</code> with a <code>static method</code> to calculate the distance between two points in the 3D space.</li>
</ul>

<h3>
<a id="user-content-problem-4-path" class="anchor" href="#problem-4-path" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 4. Path</h3>

<ul class="task-list">
<li>  Create a class <code>Path</code> to hold a sequence of points in the 3D space.</li>
<li>  Create a static class <code>PathStorage</code> with static methods to save and load paths from a text file.</li>
<li>  Use a file format of your choice.</li>
</ul>

<h3>
<a id="user-content-problem-5-generic-class" class="anchor" href="#problem-5-generic-class" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 5. Generic class</h3>

<ul class="task-list">
<li>  Write a generic class <code>GenericList&lt;T&gt;</code> that keeps a list of elements of some parametric type <code>T</code>.</li>
<li>  Keep the elements of the list in an array with fixed capacity which is given as parameter in the class constructor.</li>
<li>  Implement methods for adding element, accessing element by index, removing element by index, inserting element at given position, clearing the list, finding element by its value and <code>ToString()</code>.</li>
<li>  Check all input parameters to avoid accessing elements at invalid positions.</li>
</ul>

<h3>
<a id="user-content-problem-6-auto-grow" class="anchor" href="#problem-6-auto-grow" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 6. Auto-grow</h3>

<ul class="task-list">
<li>  Implement auto-grow functionality: when the internal array is full, create a new array of double size and move all elements to it.</li>
</ul>

<h3>
<a id="user-content-problem-7-min-and-max" class="anchor" href="#problem-7-min-and-max" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 7. Min and Max</h3>

<ul class="task-list">
<li>  Create generic methods <code>Min&lt;T&gt;()</code> and <code>Max&lt;T&gt;()</code> for finding the minimal and maximal element in the <code>GenericList&lt;T&gt;</code>.</li>
<li>  You may need to add a generic constraints for the type <code>T</code>.</li>
</ul>

<h3>
<a id="user-content-problem-8-matrix" class="anchor" href="#problem-8-matrix" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 8. Matrix</h3>

<ul class="task-list">
<li>  Define a class <code>Matrix&lt;T&gt;</code> to hold a matrix of numbers (e.g. integers, floats, decimals). </li>
</ul>

<h3>
<a id="user-content-problem-9-matrix-indexer" class="anchor" href="#problem-9-matrix-indexer" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 9. Matrix indexer</h3>

<ul class="task-list">
<li>  Implement an indexer <code>this[row, col]</code> to access the inner matrix cells.</li>
</ul>

<h3>
<a id="user-content-problem-10-matrix-operations" class="anchor" href="#problem-10-matrix-operations" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 10. Matrix operations</h3>

<ul class="task-list">
<li>  Implement the operators <code>+</code> and <code>-</code> (addition and subtraction of matrices of the same size) and <code>*</code> for matrix multiplication.</li>
<li>  Throw an exception when the operation cannot be performed.</li>
<li>  Implement the <code>true</code> operator (check for non-zero elements).</li>
</ul>

<h3>
<a id="user-content-problem-11-version-attribute" class="anchor" href="#problem-11-version-attribute" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 11. Version attribute</h3>

<ul class="task-list">
<li>  Create a <code>[Version]</code> attribute that can be applied to structures, classes, interfaces, enumerations and methods and holds a version in the format <code>major.minor</code> (e.g. <code>2.11</code>).</li>
<li>  Apply the version attribute to a sample class and display its version at runtime.</li>
</ul>

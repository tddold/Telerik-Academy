<h3>
<a id="user-content-problem-1-stringbuildersubstring" class="anchor" href="#problem-1-stringbuildersubstring" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 1. StringBuilder.Substring</h3>

<ul class="task-list">
<li>  Implement an extension method <code>Substring(int index, int length)</code> for the class <code>StringBuilder</code> that returns new <code>StringBuilder</code> and has the same functionality as <code>Substring</code> in the class <code>String</code>.</li>
</ul>

<h3>
<a id="user-content-problem-2-ienumerable-extensions" class="anchor" href="#problem-2-ienumerable-extensions" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 2. IEnumerable extensions</h3>

<ul class="task-list">
<li>  Implement a set of extension methods for <code>IEnumerable&lt;T&gt;</code> that implement the following group functions: sum, product, min, max, average.</li>
</ul>

<h3>
<a id="user-content-problem-3-first-before-last" class="anchor" href="#problem-3-first-before-last" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 3. First before last</h3>

<ul class="task-list">
<li>  Write a method that from a given array of students finds all students whose first name is before its last name alphabetically.</li>
<li>  Use LINQ query operators.</li>
</ul>

<h3>
<a id="user-content-problem-4-age-range" class="anchor" href="#problem-4-age-range" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 4. Age range</h3>

<ul class="task-list">
<li>  Write a LINQ query that finds the first name and last name of all students with age between 18 and 24.</li>
</ul>

<h3>
<a id="user-content-problem-5-order-students" class="anchor" href="#problem-5-order-students" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 5. Order students</h3>

<ul class="task-list">
<li>  Using the extension methods <code>OrderBy()</code> and <code>ThenBy()</code> with lambda expressions sort the students by first name and last name in descending order.</li>
<li>  Rewrite the same with LINQ.</li>
</ul>

<h3>
<a id="user-content-problem-6-divisible-by-7-and-3" class="anchor" href="#problem-6-divisible-by-7-and-3" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 6. Divisible by 7 and 3</h3>

<ul class="task-list">
<li>  Write a program that prints from given array of integers all numbers that are divisible by 7 and 3. Use the built-in extension methods and lambda expressions. Rewrite the same with LINQ.</li>
</ul>

<h3>
<a id="user-content-problem-7-timer" class="anchor" href="#problem-7-timer" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 7. Timer</h3>

<ul class="task-list">
<li>  Using delegates write a class <code>Timer</code> that can execute certain method at each <code>t</code> seconds.</li>
</ul>

<h3>
<a id="user-content-problem-8-events" class="anchor" href="#problem-8-events" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 8.* Events</h3>

<ul class="task-list">
<li>  Read in <a href="https://msdn.microsoft.com/en-us/library/awbftdfh.aspx">MSDN</a> about the keyword <code>event</code> in C# and <a href="https://msdn.microsoft.com/en-us/library/w369ty8x.aspx">how to publish events</a>.</li>
<li>  Re-implement the above using .NET events and following the best practices.</li>
</ul>

<h3>
<a id="user-content-problem-9-student-groups" class="anchor" href="#problem-9-student-groups" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 9. Student groups</h3>

<ul class="task-list">
<li>  Create a class <code>Student</code> with properties <code>FirstName</code>, <code>LastName</code>, <code>FN</code>, <code>Tel</code>, <code>Email</code>, <code>Marks</code> (a List), <code>GroupNumber</code>.</li>
<li>  Create a <code>List&lt;Student&gt;</code> with sample students. Select only the students that are from group number 2.</li>
<li>  Use LINQ query. Order the students by FirstName.</li>
</ul>

<h3>
<a id="user-content-problem-10-student-groups-extensions" class="anchor" href="#problem-10-student-groups-extensions" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 10. Student groups extensions</h3>

</ul>

<h3>
<a id="user-content-problem-11-extract-students-by-email" class="anchor" href="#problem-11-extract-students-by-email" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 11. Extract students by email</h3>

<ul class="task-list">
<li>  Extract all students that have email in <code>abv.bg</code>.</li>
<li>  Use <code>string</code> methods and LINQ.</li>
</ul>

<h3>
<a id="user-content-problem-12-extract-students-by-phone" class="anchor" href="#problem-12-extract-students-by-phone" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 12. Extract students by phone</h3>

<li>  Use LINQ.</li>
</ul>

<h3>

</ul>

<h3>
<a id="user-content-problem-14-extract-students-with-two-marks" class="anchor" href="#problem-14-extract-students-with-two-marks" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 14. Extract students with two marks</h3>

<ul class="task-list">

<h3>

<h3>
<a id="user-content-problem-16-groups" class="anchor" href="#problem-16-groups" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 16.* Groups</h3>

<li>  Use the <code>Join</code> operator.</li>
</ul>

<h3>
<a id="user-content-problem-17-longest-string" class="anchor" href="#problem-17-longest-string" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 17. Longest string</h3>

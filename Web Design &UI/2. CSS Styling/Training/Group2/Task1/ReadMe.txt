Аз лично ползвам LiveReload, за да ми се презарежда автоматично страницата, когато save-вам работата си в IDE-то (тъй като пиша на WebStorm и не мога да се възползвам от тази благинка на Web Essentials за Visual Studio).
Как можете да го направите и вие:
1. Сваляте и инсталирате Git (Най-вероятно вече го имате инсталирано, ако сте си качвали домашните в GitHub).
2. Сваляте и инсталирате NodeJS.
3. Инсталирате LiveReload Chrome Extension.
4. Сваляте и разархивирате този архив, който си подготвих за изпита.
След като имате инсталирани тези четири неща, влизате в папката, която сте създали след разархивирането на горния архив. Там си слагате .html, .css и папката с картинките, по които работите.
След това отваряте конзола в текущата папка и пишете следните команди:
npm install -g grunt-cli
npm install
grunt
Ако всичко мине успешно, в конзолата (която трябва да стои отворена през цялото време, докато работите!) ще изпише, че е стартиран task watch. Това ще следи за промени по файловете и ще рефрешва браузъра при всеки save.
Обръщам внимание, че index.html файлът трябва да бъде стартиран през web server (През WebStorm е най-лесно, като просто се натисне иконката на Chrome. Със Sublime не знам как точно се подкарва на web server, но можете да инсталирате Phyton и да си потърсите командата).
След като e стартиран index.html файлът в Chrome, трябва да се натисне LiveReload Extension иконката, която е в Chrome (горе вдясно). Ако всичко мине както трябва, кръгчето в средата ще се оцвети в черно и при всяка запазена промяна на файл (.css или .html), ще се refresh-ва автоматично browser-a.

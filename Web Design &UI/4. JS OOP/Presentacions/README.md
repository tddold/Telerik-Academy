    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1><a id="user-content-javascript-oop" class="anchor" href="#javascript-oop" aria-hidden="true"><span class="octicon octicon-link"></span></a>JavaScript-OOP</h1>

<p>JavaScript OOP Course Repository</p>

<h2><a id="user-content-preparing-the-local-machine-for-unit-testing-with-mocha-and-chai" class="anchor" href="#preparing-the-local-machine-for-unit-testing-with-mocha-and-chai" aria-hidden="true"><span class="octicon octicon-link"></span></a>Preparing the local machine for Unit testing with Mocha and Chai</h2>

<ul>
<li>  Install <a href="https://nodejs.org/" title="Node.js">Node.js</a>

<ul>
<li>  Try if it is working by typing in CMD/Terminal <code>$ node -v</code> (should produce result)</li>
</ul></li>
<li>  Open CMD/Terminal and run <code>$ npm install -g mocha</code></li>
</ul>

<h2><a id="user-content-preparing-for-the-tests-for-each-homework" class="anchor" href="#preparing-for-the-tests-for-each-homework" aria-hidden="true"><span class="octicon octicon-link"></span></a>Preparing for the tests for each homework</h2>

<ul>
<li>  Checkout the repository for the particular homework </li>
<li>  Open CMD/Terminal and navigate to the checked out repository with the homework</li>
<li>  Run <code>npm install</code> in CMD/Terminal

<ul>
<li>  A folder <code>node_modules</code> should appear</li>
</ul></li>
<li>  You are ready to run the tests</li>
</ul>

<h2><a id="user-content-running-the-tests" class="anchor" href="#running-the-tests" aria-hidden="true"><span class="octicon octicon-link"></span></a>Running the tests</h2>

<ul>
<li>  Navigate to the folder of the particular homework in CMD/Terminal</li>
<li>  Requirements:

<ul>
<li>  JavaScript files must be called task-1.js, task-2.js etc..</li>
<li>  Each .js file must contain <code>module.exports=[name of the object/function]</code></li>
</ul></li>
<li>  Run <code>npm test</code>

<ul>
<li>  Test results should appear on the CMD/Terminal</li>
</ul></li>
</ul>
